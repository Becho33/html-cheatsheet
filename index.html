HTML ELEMENTS

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table head The set of rows defining the column headers in a table.
<body> Table body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container..
<foot> Table footer The set of rows defining the footer in a table.
<a> hyperlink This is a hyperlink, remember to include href= and the target argument. also include the display text .
target=_blank new window This opens hyperlink in new window
<img> Image You will need to include "src=", width and hight
<h1-6> Headers Biggest to smallest headers.
<p> paragraph This denotes the paragraph.
